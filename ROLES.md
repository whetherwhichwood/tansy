## Collaboration Workflow

### Daily Standups (15 minutes)
- **When:** 9:00 AM EST
- **Format:** What did you do yesterday? What are you doing today? Any blockers?
- **Participants:** All 3 team members

### Sprint Planning (1 hour)
- **When:** Every 2 weeks
- **Format:** Review backlog, estimate tasks, assign work
- **Participants:** All 3 team members

### Code Reviews
- **Process:** All code changes require review before merging
- **Frontend Developer:** Reviews frontend changes
- **Backend Developer:** Reviews backend changes
- **Product Manager:** Reviews business logic and requirements

### Communication Channels
- **Primary:** GitHub Issues and Pull Requests
- **Secondary:** Slack/Discord for quick questions
- **Meetings:** Weekly 1-on-1s with Product Manager

# Neighborhood Micro-Favor App

A community-driven platform for neighbors to help each other with small tasks and favors.

## Development

```bash
npm run dev
```

Runs on http://localhost:3004

## Features (Planned)

- User authentication and profiles
- Favor request/offer system
- Location-based matching
- Rating and review system
- In-app messaging
- Task categories and filtering

## Tech Stack

- Next.js 15.5.6
- TypeScript
- Tailwind CSS
- React 19

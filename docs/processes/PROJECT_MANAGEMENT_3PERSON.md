## Daily Workflow

### Daily Standups (15 minutes)
- **Time**: 9:00 AM EST
- **Participants**: All 3 team members
- **Format**: 
  - What did you do yesterday?
  - What are you doing today?
  - Any blockers or dependencies?

### Code Review Process
1. **Developer** creates feature branch and implements changes
2. **Other Developer** reviews code for quality and correctness
3. **Product Manager** reviews for business requirements compliance
4. **Merge** to main after all reviews complete

### Weekly 1-on-1s
- **Product Manager ↔ Frontend Developer**: Monday 2:00 PM
- **Product Manager ↔ Backend Developer**: Tuesday 2:00 PM
- **Format**: Progress review, feedback, planning, blockers
